## How to deploy

Please use python3.

```sh
git clone https://github.com/bohyun-jung/bit-fall.git
cd R001
pip install -r requirements.txt
./run.sh
```

* Visit http://localhost:10090/
* You can send a message from http://localhost:10090/post
